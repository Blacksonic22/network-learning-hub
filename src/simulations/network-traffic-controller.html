Network Traffic Controller
Interactive CS mini-game

Send one packet at a time. Prevent collisions.

Tap a device to transmit traffic across the shared network. Single, well-timed sends earn points. Overlapping sends trigger a collision and push the network into an error state.

How to play

Use the three endpoints like stations on a shared channel. Only one sender should be active during a transmission window.

  • Router A starts a top-lane transfer.
  • Router B transmits through the center node.
  • Router C injects traffic from the lower lane.
  • If two devices fire before the first finishes, a collision occurs.
