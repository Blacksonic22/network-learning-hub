    What you are seeing

    • Carrier sense: a host checks whether the cable looks busy before sending.
    • Collision detection: overlapping transmissions reveal that two hosts started too close together in time.
    • Jam signal: both hosts broadcast a short warning so every device knows the frame was corrupted.
    • Binary exponential backoff: each sender waits a random number of slot times before retrying, lowering the chance of repeating the same collision.
